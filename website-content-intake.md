# Website Content Intake

## Company basics
- Company name (legal/trading)
- Tagline / short slogan
- Year founded
- Years of experience
- Number of completed projects
- Number of team members
- Awards / certifications (name + year)
- Service area (cities, regions, countries)

## Contact
- Primary phone
- Secondary phone (optional)
- Public email
- Office address
- Office hours
- Google Maps link or coordinates
- Facebook URL
- Instagram URL
- LinkedIn URL
- YouTube / Vimeo URL
- WhatsApp number 

## About
- Mission statement (1–2 sentences)
- Vision statement (1–2 sentences)
- Brand story / company history (150–300 words)
- Core values (3–6, one per line)
- What makes us different (3–5 bullets)
- Team --

## Services (repeat per service)
- Service name - Category (residential, commercial, interior, etc.) ...
- Short description (1–2 sentences for cards)
- Full description (150–250 words for detail page)
- Any additional services beyond the main list
